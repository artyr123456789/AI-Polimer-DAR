### README для проекта AI Polymer - DAR

#### Название Проекта
**AI Polymer DAR**

Краткое описание: 
AI Polymer — это искусственный интеллект, предназначенный для определения материалов, из которых изготовлен полимер, на основе свойств конечного продукта. Это решение может быть использовано в научных исследованиях, производстве и разработке новых полимерных материалов.
DAR - это 2 модель искусственного интеллекта, предназначенного для определения конечных свойств полимера, полученного из введенных материалов. Это решение может быть использовано в научных исследованиях, производстве и разработке новых полимерных материалов.

#### Назначение
Проект направлен на решение обратной задачи в области полимерных материалов: определение исходных компонентов (материалов) на основе известных свойств конечного продукта. Такой подход помогает ускорить процесс разработки новых материалов и оптимизировать существующие процессы производства [[2]].

---

#### Основные Возможности
- Прогнозирование состава полимера на основе его физико-химических свойств.
- Учет различных условий производства (температура, давление, оборудование, ход реакции и время реакции).
- Интеграция базы данных схожести материалов для подбора заменителей.
- Гибкая настройка параметров модели через конфигурационные файлы.
- Веб-интерфейс для удобного использования без глубоких технических знаний(важно).

---

#### Требования
- Python 3.8+
- PyTorch или TensorFlow + keras для обучения моделей
- Docker для контейнеризации приложения
- (SQL)SQLite или PostgreSQL для хранения данных о материалах и их свойствах
- Библиотеки для обработки данных: pandas, numpy, scikit-learn

#### Использование
1. Откройте веб-интерфейс по адресу `http://localhost:5000`.
2. Введите свойства полимера (например, плотность, температура плавления, электропроводность).
3. Получите список возможных материалов и условий их производства.

Пример запроса через API:
```bash
curl -X POST http://localhost:5000/predict \
     -H "Content-Type: application/json" \
     -d '{"density": 0.92, "melting_point": 120, "conductivity": 0.3}'
```

#### Архитектура Модели
Проект использует два вариационных автокодировщика (VAE):
1. **Forward VAE:** Предсказывает свойства полимера на основе материалов и условий производства.
2. **Inverse VAE:** Определяет материалы и условия производства на основе заданных свойств полимера.

Модель состоит из нескольких входных слоев для анализа различных переменных (пропорции, температура, давление), //шести скрытых слоев и нескольких выходных слоев для предсказания результатов [[1]].

#### Данные
Данные должны содержать информацию о материалах, их свойствах и условиях производства. Формат данных:
- `polymer_properties.csv`: Свойства полимеров (плотность, температура плавления, теплопроводность и т.д.).
- `reaction_conditions.json`: Условия производства (температура, давление, время реакции).
- `material_embeddings.pkl`: Предобученные эмбеддинги материалов для быстрого поиска схожести.

Для генерации синтетических данных можно использовать скрипт:
```bash
python data/generate_simulated_data.py
```
